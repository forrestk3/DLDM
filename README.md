# emacs常用命令
## 代码折叠
这里说的是emacs自带的HideShow mode.

进入HideShow mode： M-x hs-minor-mode（幸亏有tab键。。要不这么长的命令=。=）

主要的功能：

* C-c @ C-M-s 显示所有的代码
* C-c @ C-M-h 折叠所有的代码
* C-c @ C-s 显示当前代码区
* C-c @ C-h 折叠当前代码区
* C-c @ C-c 折叠/显示当前代码区

## 字体大小修改
放大字体: Ctrl-x Ctrl-+ 或 Ctrl-x Ctrl-=
缩小字体: Ctrl-x Ctrl–
重置字体: Ctrl-x Ctrl-0
